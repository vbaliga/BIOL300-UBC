<h3>Example 17.1. <a href="../wp-content/data/chapter17/chap17e1LionNoses.csv">Lion noses</a></h3>
<em><b>Estimate a linear regression</b> and <b>calculate predicted values and their uncertainty</b> for the relationship between lion age and proportion of black on the lion nose</em>.

Read and inspect the data.
<pre>lion &lt;- read.csv(url("http://www.zoology.ubc.ca/~schluter/WhitlockSchluter/wp-content/data/chapter17/chap17e1LionNoses.csv"))
head(lion)
</pre>
Basic <strong>scatter plot</strong>.
<pre>plot(ageInYears ~ proportionBlack, data = lion)
</pre>
Commands for a fancier scatter plot using more options (Figure 17.1-1) are [tippy title="here" width = "680" header = "off"]
<pre>plot(ageInYears ~ proportionBlack, data = lion, pch = 16, col="firebrick", 
	las = 1, cex = 1.5, bty = "l", xlim = c(0,0.8), ylim = c(0,14), 
	ylab = "Age (years)", xlab = "Proportion black")
</pre>
[/tippy].

<strong>Fit the linear regression to the data using least squares</strong>. Use <code>lm</code>, which was also used for ANOVA. Save the results into a model object, and then use other commands to extract the quantities we want.
<pre>lionRegression &lt;- lm(ageInYears ~ proportionBlack, data = lion)
</pre>
Show the <strong>estimated linear regression coefficients: slope and intercept</strong>. The output of <code>summary</code> includes other valuable quantities, including standard errors and a <i>t</i>-test of zero regression slope. If you want only the regression coeffients without the rest of the data, use <code>coef(lionRegression)</code>.
<pre>summary(lionRegression)
</pre>
<strong>Add the least squares regression line to the plot</strong>. The quick method is to use <code>abline</code>. If needed, redraw the scatter plot first.
<pre>abline(lionRegression)
</pre>
Another way to draw the regression line is to generate predicted values and then connect them with a line (Figure 17.1-4). This has the advantage of more control - for example, we can make sure that the line doesn't fall outside the range of the data, avoiding extrapolation. Commands to accomplish this are [tippy title="here" width = "680" header = "off"]
<pre>xpts &lt;- range(lion$proportionBlack)
ypts &lt;- predict(lionRegression, data.frame(proportionBlack = xpts))
lines(ypts ~ xpts, lwd = 1.5)
</pre>
[/tippy].

<strong>Use the regression line for prediction</strong>. For example, here is how to predict mean lion age corresponding to a value of 0.50 of proportion black in the nose. Standard error of the predicted mean is included.
<pre>predict(lionRegression, data.frame(proportionBlack = 0.50), se.fit = TRUE)
</pre>
<strong>Calculate the residual and regression mean squares</strong>. All the mean squares are included in the anova table, which is generate using the <code>anova</code> command.
<pre>anova(lionRegression)
</pre>
<strong>Standard error of slope</strong>. The standard errors are provided in the output of the <code>summary</code> function.
<pre>summary(lionRegression)
</pre>
<strong>95% confidence interval for slope</strong>.
<pre>confint(lionRegression)
</pre>
<strong>95% confidence bands</strong> (Figure 17.2-1). If needed, redraw the scatter plot, including the regression line, before executing the commands below. The <code>seq</code> function below generates a sequence of 100 new x-values evenly spaced between the smallest and largest values of <code>proportionBlack</code>. The <code>predict</code> function calculates the corresponding predicted mean y-values on the regression line, as well as the lower and upper limits of the confidence intervals for the predicted mean y-values at each x-value (these are put into the data frame named <code>ypt</code>. Finally, the <code>lines</code> functions connect the consecutive lower limits of the confidence intervals with a line, and do the same for the consecutive upper limits, forming the bands.
<pre>xpt &lt;- seq(min(lion$proportionBlack), max(lion$proportionBlack), 
	length.out = 100)
ypt &lt;- data.frame( predict(lionRegression, 
	newdata = data.frame(proportionBlack = xpt), 
	interval = "confidence") )
lines(ypt$lwr ~ xpt, lwd = 1.5, lty = 2)
lines(ypt$upr ~ xpt, lwd = 1.5, lty = 2)
</pre>
<strong>95% prediction intervals</strong> (Figure 17.2-1). If needed, redraw the scatter plot, including the regression line, before executing the commands below. The commands below are the same as those used to generate the confidence bands (see above), but here the lower and upper limits are confidence intervals for the prediction of a single individual y-value, rather than the mean value, corresponding to each x-value.
<pre>xpt &lt;- seq(min(lion$proportionBlack), max(lion$proportionBlack), 
	length.out = 100)
ypt &lt;- data.frame(predict(lionRegression, newdata = data.frame(proportionBlack = xpt), 
	interval = "prediction", level = 0.95))
lines(ypt$lwr ~ xpt, lwd = 1.5, lty = 3)
lines(ypt$upr ~ xpt, lwd = 1.5, lty = 3)
</pre>

<h3>Example 17.3. <a href="../wp-content/data/chapter17/chap17e3PlantDiversityAndStability.csv">Prairie home campion</a></h3>
<em><b>Test the null hypothesis of zero regression slope</b>. The data are from an experiment investigating the effect of plant species diversity on the stability of plant biomass production</em>.

Read and inspect the data.
<pre>prairie &lt;- read.csv(url("http://www.zoology.ubc.ca/~schluter/WhitlockSchluter/wp-content/data/chapter17/chap17e3PlantDiversityAndStability.csv"))
head(prairie)
</pre>
<strong>Log-transform</strong> stability and include the new variable in the data frame. Inspect the data frame once again to make sure the function worked as intended.
<pre>prairie$logStability &lt;- log(prairie$biomassStability)
head(prairie)
</pre>
<strong>Scatter plot with regression line</strong>.
<pre>plot(logStability ~ nSpecies, data = prairie) 
prairieRegression &lt;- lm(logStability ~ nSpecies, data = prairie)
abline(prairieRegression)
</pre>
Commands for a fancier scatter plot with more options (Figure 17.3-1) are [tippy title="here" width = "680" header = "off"]
<pre>plot(logStability ~ nSpecies, data = prairie, bty = "l", col = "firebrick", 
	pch = 1, las = 1, cex = 1.5, xlim = c(0,16), xaxp = c(0,16,8), 
	xlab = "Species number treatment", ylab = "Log-transformed ecosystem stability")
prairieRegression &lt;- lm(logStability ~ nSpecies, data = prairie)
xpts &lt;- c(1,16)
ypts &lt;- predict(prairieRegression, newdata = data.frame(nSpecies = xpts))
lines(ypts ~ xpts, lwd=1.5)</pre>
[/tippy].

<strong><i>t</i>-test and 95% confidence interval of regression slope</strong>.
<pre>prairieRegression &lt;- lm(logStability ~ nSpecies, data = prairie)
summary(prairieRegression)
confint(prairieRegression)
</pre>
<strong>ANOVA test of zero regression slope</strong> (Table 17.3-2).
<pre>anova(prairieRegression)
</pre>
<strong><i>R</i><sup>2</sup></strong>. The <i>R</i><sup>2</sup> associated with the regression is part of the output of the <code>summary</code> command.
<pre>summary(prairieRegression)$r.squared
</pre>

<h3>Figure 17.8-1. <a href="../wp-content/data/chapter17/chap17f8_1IronAndPhytoplanktonGrowth.csv">Iron and phytoplankton growth</a></h3>
<em><b>Fit a nonlinear regression curve having an asymptote</b> (Michaelis-Menten curve). The data are the relationship between population growth rate of a phytoplankton in culture and the concentration of iron in the medium</em>.

Read and examine data.
<pre>phytoplankton &lt;- read.csv(url("http://www.zoology.ubc.ca/~schluter/WhitlockSchluter/wp-content/data/chapter17/chap17f8_1IronAndPhytoplanktonGrowth.csv"))
head(phytoplankton)
</pre>
<strong>Scatter plot</strong>.
<pre>plot(phytoGrowthRate ~ ironConcentration, data = phytoplankton)
</pre>
Instructions for a scatter plot with more options are [tippy title="here" width = "680" header = "off"]
<pre>plot( phytoGrowthRate ~ ironConcentration, data = phytoplankton, pch = 16, 
	col = "firebrick", las = 1, cex = 1.5, bty = "l", 
	ylab = "Growth rate (no./day)",
	xlab = expression(paste("Iron concentration (", mu, "mol)")) )
</pre>
[/tippy].

<strong>Fit a Michaelis-Menten curve</strong> to the phytoplankton data using the <code>nls</code> (nonlinear least squares). To fit the curve, provide a formula that also includes symbols (letters) for the parameters to be estimated. In the following function, we use "a" and "b" to indicate the parameters of the Michaelis-Menten curve we want to estimate. The function includes an argument where we must provide an initial guess of parameter values. The value of the initial guess is not so important -- here we choose a=1 and b=1 as initial guesses. The first function below carried out the model fit and save the results in an R object named <code>phytoCurve</code>.
<pre>phytoCurve &lt;- nls(phytoGrowthRate ~ a*ironConcentration / ( b+ironConcentration), 
	data = phytoplankton, list(a = 1, b = 1))
</pre>
<strong>Obtain the parameter estimates</strong> using the <code>summary</code> command to , including standard errors and <i>t</i>-tests of null hypotheses that parameter values are zero.
<pre>summary(phytoCurve)
</pre>
<strong>Add the nonlinear regression curve to scatter plot</strong>. If necessary, redraw the scatter plot before issuing the following commands.
<pre>xpts &lt;- seq(min(phytoplankton$ironConcentration), 
	max(phytoplankton$ironConcentration), length.out = 100)
ypts &lt;- predict(phytoCurve, new = data.frame(ironConcentration = xpts))
lines(xpts, ypts)
</pre>
Many of the functions that can be used to extract results from a saved <code>lm</code> object work in the same way when applied to an <code>nls</code> object, such as <code>predict</code>, <code>residuals</code>, and <code>coef</code>.

<strong>Fit a logistic regression</strong>.
<pre>guppyGlm &lt;- glm(mortality ~ exposureDurationMin, data = guppy,
	family = binomial(link = logit))
</pre>
<strong>Add logistic regression curve to scatter plot</strong>.
<pre>xpts &lt;- seq(min(guppy$exposureDurationMin), max(guppy$exposureDurationMin), 
	length.out = 100)
ypts &lt;- predict(guppyGlm, newdata = data.frame(exposureDurationMin = xpts), 
	type = "response")
lines(xpts, ypts)
</pre>
<strong>Table of regression coefficients</strong>, with parameter estimates (Table 17.9-2).
<pre>summary(guppyGlm)
</pre>
<strong>95% confidence intervals for parameter estimates</strong>. It is necessary to load the <code>MASS</code> package first.
<pre>library(MASS)
confint(guppyGlm)
</pre>
<strong>Predict probability of mortality</strong> (mean mortality) for a given x-value, 10 min duration, including standard error of prediction.
<pre>predict(guppyGlm, newdata = data.frame(exposureDurationMin = 10),
	type = "response", se.fit = TRUE)
</pre>
<strong>Estimate the LD50</strong>, the dose at which half the individuals are predicted to die from exposure.
